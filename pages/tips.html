憑證服務操作說明

步驟指令
0. FQDN Azure DNS zones Recordsets: add a CNAME record '_acme-challenge.<your-domain>' Alias {{.FQDN}}
1. 帳號檢查 curl "{{.BaseURL}}/register?account={{.ACME_ACCOUNT}}"
2. 憑證發行 curl -X POST "{{.BaseURL}}/cert?domain=*.itsower.com.tw"
3. 憑證下載 curl -OJ "{{.BaseURL}}/cert?domain=*.itsower.com.tw"
4. 憑證狀態 curl "{{.BaseURL}}/health?domain=*.itsower.com.tw"
用於判斷是否可以下載憑證 (status: OK >30天 / WARN ≤30天 / ERROR 已過期)
5. 檢查到期 curl "{{.BaseURL}}/expire?domain=*.itsower.com.tw"
6. 憑證更新特定憑證 curl -X POST "{{.BaseURL}}/renew?domain=*.itsower.com.tw"
憑證中心 --force 更新憑證，需搭配 3. 重新下載憑證
7. 憑證更新所有憑證 curl -X POST "{{.BaseURL}}/renew"
憑證中心 --force 更新所有憑證，需搭配 3. 重新下載憑證
